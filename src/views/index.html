Typography

All typography and vertical rhythm uses base HTML elements (h1, h2, p, table, etc.) to set styles. The Sass module controlling this is a custom module found in /src/assets/sass/base/_typesetter.scss

Color

Base colors are managed as variables and can be found in /src/assets/sass/helpers/_variables.scss
